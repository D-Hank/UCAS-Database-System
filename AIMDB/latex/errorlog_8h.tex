\hypertarget{errorlog_8h_Description}{}\doxysubsection{Description}\label{errorlog_8h_Description}
This file provides the error handling and logging utility.


\begin{DoxyEnumerate}
\item error code The error code is a decimal number with 8 digits\+:

FFFLLLLL
\end{DoxyEnumerate}

The higher 3 digits show the source file ID, while the lower 5 digits indicate the line number in the source file, where the error occurs.

The following macros are useful\+: \begin{DoxyVerb} EL_GET_FILEID(err_code)
 EL_GET_LINENO(err_code)
 EL_GET_FILENAME(err_code)      
\end{DoxyVerb}

\begin{DoxyEnumerate}
\item initialize
\end{DoxyEnumerate}

(1) \mbox{\hyperlink{runaimdb_8cc_a0ddf1224851353fc92bfbff6f499fa97}{main()}} must call \mbox{\hyperlink{classErrorLog_a62760139b8a7cc8075e3e1f5477e1207}{Error\+Log\+::init(int level, const char $\ast$logfile)}};

(2) then every thread must new an \mbox{\hyperlink{classErrorLog}{Error\+Log}} instance\+: \begin{DoxyVerb}thread_el= new ErrorLog("thread_name");
\end{DoxyVerb}
 (3) put source file names into EL\+\_\+src\+\_\+file\+\_\+name\mbox{[}\mbox{]} in Error\+Log.\+cc


\begin{DoxyEnumerate}
\item normal use

\mbox{\hyperlink{errorlog_8h_ac17898c0440b632bf2b19f799cba971e}{EL\+\_\+\+LOG\+\_\+\+DEBUG(format, args, ...)}}; \mbox{\hyperlink{errorlog_8h_a8ccd5d225dd561a2c66b9d50e064eb79}{EL\+\_\+\+LOG\+\_\+\+INFO(format, args, ...)}}; \mbox{\hyperlink{errorlog_8h_aa0ee3b27dc5c3db4278bcd5278aa2b16}{EL\+\_\+\+LOG\+\_\+\+WARN(format, args, ...)}}; \mbox{\hyperlink{errorlog_8h_aeac6c9620098692e9496de0d95bf12fe}{EL\+\_\+\+LOG\+\_\+\+ERROR(format, args, ...)}}; \mbox{\hyperlink{errorlog_8h_aae923b24c8a95a7ca9b55f9751f401bc}{EL\+\_\+\+LOG\+\_\+\+SERIOUS(format, args, ...)}};

The message will be written into the specified log file as follows

\mbox{[}thread\mbox{]}\mbox{[}level\mbox{]}\mbox{[}\href{file:lineno}{\texttt{ file\+:lineno}}\mbox{]} message a call stack trace will be shown for error and serious messages.

Moreover, an assertion can be written as\+:

\mbox{\hyperlink{errorlog_8h_ac7f7db37ada929c3be8e78703e10a83c}{EL\+\_\+\+ASSERT(expression)}};

The expression is a test that evaluates to a True or False value. If the value is False, then a debug assertion message will be generated and written to the log.
\item In a worker thread\+:

(1) reset and clear the error messages

\mbox{\hyperlink{errorlog_8h_acdf5f2d54ad4457776d64007e479cae4}{EL\+\_\+\+RESET()}};

(2) then follow the above to log messages

(3) finally, obtain error code and message as follows

int err\+\_\+code= \mbox{\hyperlink{errorlog_8h_a61216e44f2ef8a4d5a399ca90f4b5f9a}{EL\+\_\+\+ERRCODE()}}; const char $\ast$err\+\_\+msg= \mbox{\hyperlink{errorlog_8h_a9d4fe9da81a97503fae922ba8c7e0f4f}{EL\+\_\+\+ERRMSG()}};

(4) optionally, flush the log file

\mbox{\hyperlink{classErrorLog_a5ad2be67100d49097713357937f54c47}{Error\+Log\+::flush\+Log()}};
\item Before exiting, call the following globally once

\mbox{\hyperlink{classErrorLog_a764e480ee5b420c5be96ceaa732f407b}{Error\+Log\+::close\+Log()}}; 
\end{DoxyEnumerate}
